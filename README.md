# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1UgGpN1ckRwU6NpQmWo0eM9W7zTM8c4Wk

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

### Quick commands (PowerShell)

Open PowerShell in the project root and run the following commands:

```powershell
cd "C:\Users\Lenovo\Downloads\Student-Institute-smart-card-main\Student-Institute-smart-card"
npm install
# Start dev server (default: http://localhost:3000)
npm run dev

# Build production bundle
npm run build

# Preview the production build (default: http://localhost:4173)
npm run preview
```
